## What You'll Learn

- What **NumPy** is and why it's the foundation of data science in Python
- What **Pandas** is and how to use it to work with tabular data
- How to **explore a real-world dataset** step by step
- Basic data cleaning, inspection, and analysis techniques

## Getting Started

### Option 1 — Open in Google Colab (Recommended, no setup needed)

[![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/1nIfO1w4hLGTwUl3IWVt8P7hRg1EZuPSz?usp=sharing)

### Option 2 — Run Locally

**1. Clone the Repository**
```bash
git clone https://github.com/YOUR-USERNAME/YOUR-REPO-NAME.git
cd YOUR-REPO-NAME
```

**2. Install Dependencies**
```bash
pip install numpy pandas matplotlib jupyter
```

**3. Launch the Notebook**
```bash
jupyter notebook Data_Analysis_Workshop_Beginner.ipynb
```
